# Bite Sites Template Skeleton

This is a neutral starter template for new restaurant sites.

## Includes

- `index.html` with shared section IDs and `data-i18n` hooks
- `assets/js/main.js` for navigation, scroll-spy, and open/closed badge behavior
- `assets/js/i18n.js` with DE/EN placeholder translations
- `assets/js/menu.js` with placeholder menu data + renderers
- `assets/css/styles.css` with a simple neutral visual baseline

## Usage

1. Copy this folder to a new restaurant repo.
2. Replace placeholder text, contact details, hours, and menu data.
3. Update design tokens and typography in `assets/css/styles.css`.
